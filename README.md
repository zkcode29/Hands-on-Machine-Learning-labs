# Hands-on-Machine-Learning-labs

This repository contains **hands-on laboratory work, assignments, and exams** designed to build a strong foundation in **Machine Learning**, from basic data handling to advanced neural network models.

The focus is on **conceptual clarity, mathematical understanding, and practical implementation using Python**.

---

##  Repository Overview

The repository is organized into:
- **Laboratory Exercises**
- **Assignments**
- **Lab Exams**
- **Unsupervised Machine Learning**

Each lab builds progressively on previous concepts.
